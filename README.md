# Operant
Centro de Informações dos Técnicos de Campo – KUHN do Brasil
O Operant é a plataforma oficial utilizada para gestão, acompanhamento e controle das atividades dos técnicos de campo da KUHN do Brasil.
O sistema centraliza informações operacionais, administrativas e de conformidade, garantindo eficiência e rastreabilidade dos processos.

# Estrutura do Sistema
O sistema está organizado em duas categorias principais:

# Administração
Área destinada à gestão e supervisão das atividades dos técnicos de campo.
Funcionalidades


# Análise por Técnico
Permite visualizar o panorama individual dos checklists realizados, facilitando o acompanhamento de desempenho e conformidade.


# Estatísticas
Apresenta dados em formato gráfico, possibilitando a análise de:

Ferramentas com maior índice de falhas;
Técnicos com pendência no envio de checklists;
Itens com maior demanda de compra.



# Maletas
Acesso às imagens das maletas e veículos dos técnicos, para verificação de organização e conformidade.


# Aprovação de Compras
Responsável pela análise e aprovação das solicitações de aquisição de ferramentas.


# Documentação
Armazenamento e gestão dos documentos dos técnicos.


# Regras
Diretrizes e responsabilidades administrativas dentro do sistema.


Responsabilidades do Administrador

Acompanhar continuamente o envio dos checklists conforme o cronograma estabelecido;
Realizar a verificação e tratativa de ocorrências, irregularidades e não conformidades;
Assegurar o fechamento mensal das maletas após validação dos registros;
Monitorar pendências e o status operacional dos técnicos;
Garantir a integridade, segurança e confidencialidade das informações do sistema.


# Técnicos
Área destinada à execução das atividades operacionais pelos técnicos de campo.
Funcionalidades


# Checklist
Registro e verificação das condições das ferramentas.


# Compras
Solicitação de aquisição de ferramentas, incluindo demandas urgentes.


# Documentação
Acesso aos documentos pessoais e profissionais.


# Regras
Normas e diretrizes obrigatórias para atuação no sistema.


Diretrizes para Utilização

O checklist trimestral deve ser preenchido até o dia 5, conforme prazo estabelecido;
É obrigatório anexar registro fotográfico da caixa de ferramentas organizada no checklist;
Ocorrências, inconsistências ou irregularidades devem ser detalhadas no campo de observações;
O não cumprimento do prazo pode impactar o processo de reposição de ferramentas;
Recomenda-se manter a caixa de ferramentas organizada, conforme diretrizes, para facilitar a validação.


✅ Objetivo
O Operant tem como objetivo garantir maior controle operacional, padronização dos processos e aumento da eficiência na gestão das ferramentas e atividades dos técnicos de campo.
